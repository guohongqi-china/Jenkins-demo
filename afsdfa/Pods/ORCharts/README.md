# <a id="installation"></a>Installation

#### ORCharts
```ruby
pod 'ORCharts'
```
#### Only Ring

```ruby
pod 'ORCharts/Ring'  
```
#### Only Line

```ruby
pod 'ORCharts/Line'  
```

# <a id="example"></a>Example 

To run the example project, clone the repo, and run `pod install` from the Example directory first.
